# CrimeWave
project for steg

CrimeWave is a C program that will use echo hiding to encode a message of any data type (text, image, etc ..) into 16-bit audio files (wav, mono). 

It will use WaveReader, an audio file parser, to read in wave files and grab their header info. WaveReader is a program written by our professor, John Ortiz.
